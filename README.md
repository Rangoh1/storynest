# StoryNest

StoryNest is a web platform where people can share real-life experiences through usernames while maintaining their privacy.

## Features

- Read stories without signing in
- Create an account
- Publish stories
- Comment on stories
- React to stories
- Browse stories by category
- Admin approval before publishing

## Technologies

- HTML5
- CSS3
- JavaScript
